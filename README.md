# Octoparency

A modern web application built with HTML, CSS, and JavaScript.

## Project Structure

```
octoparency/
├── assets/
│   ├── css/
│   │   ├── styles.css
│   │   └── error.css
│   ├── js/
│   │   └── script.js
│   └── images/
│       └── ocular-logos/
│           ├── logo1.png
│           ├── logo2.png
│           └── ...
├── pages/
│   └── 404.html
├── index.html
├── vercel.json
└── package.json
```

## Development

1. Clone the repository
2. Run `npm install` 
3. Run `npm start` to start the local development server
